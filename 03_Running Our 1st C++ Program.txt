# Syntax for Compiling & Running Your C++ Code File Using Terminal in Windows:

 
* Compile C++ code file using this statement (g++ "Name of File.cpp") 

* After compilation of C++ code file we will get a file named "a.exe", this is a "executable file" which can run on our "Operating System". 

* To cutomize name of this file .exe file while compiling the C++ comand, compile C++ code file using this statement (g++ "Name of File.cpp" -o "Type here customize name that you want of your .exe file") 

* In Windows you can run the compiled .exe file using this statement (.\a.ex) or (.\"Whater you're exe's name is".exe)
